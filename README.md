# Buddy's vim files

## Installation

### Create a new Configuration
~~~ bash
$ cd ~/
$ git clone https://github.com/buddylindsey/vim.git .vim
$ ln -s .vim/vimrc .vimrc
~~~

### Overwriting Exisitng Configuration
~~~ bash
$ cd ~/ && rm -rf .vim && rm .vimrc
$ git clone https://github.com/buddylindsey/vim.git .vim
$ ln -s .vim/vimrc .vimrc
~~~
